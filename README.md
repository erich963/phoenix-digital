# Phoenix Digital

B2B lead generation systems built on Google Ads, fast websites, and practical strategy.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Heroicons

## Deployment

Deploy to Vercel with zero configuration:

```bash
npx vercel
```

## Contact Form

The contact form currently logs submissions to the console. To enable real form submissions, integrate a backend service (e.g., Resend, SendGrid, or a Next.js API route).
